📦 ListaComArray

Implementação de uma lista dinâmica em Java baseada em arrays, desenvolvida com o objetivo de praticar conceitos fundamentais de estrutura de dados, manipulação de memória e lógica de baixo nível semelhante ao funcionamento interno de uma ArrayList.

🚀 Sobre o projeto

Este projeto consiste na criação de uma estrutura de dados do tipo lista, utilizando um array como base. A implementação contempla operações essenciais como inserção, remoção, busca e atualização de elementos, além de controle de capacidade e redimensionamento automático.

A lista pode operar tanto em modo redimensionável quanto fixo, permitindo simular diferentes comportamentos de estruturas reais.

⚙️ Funcionalidades

✔ Inserção de elementos (add)

✔ Inserção por índice (add(index, element))

✔ Remoção por índice (remove)

✔ Remoção por valor (removeFirst)

✔ Acesso a elementos (get)

✔ Atualização de valores (set)

✔ Verificação de existência (contains)

✔ Busca de índice (indexOf, lastIndexOf)

✔ Conversão para array (toArray)

✔ Limpeza da lista (clear)

✔ Controle de tamanho (size)

✔ Verificação de estado (isEmpty, isFull)

✔ Redimensionamento automático (resize)

🧠 Conceitos aplicados

Estruturas de dados (listas e arrays)

Complexidade de algoritmos (O(n))

Manipulação de índices

Alocação e realocação de memória

Encapsulamento e organização de código

Boas práticas em Java

📌 Objetivo

O principal objetivo deste projeto é consolidar o entendimento sobre como estruturas de dados funcionam internamente, sem depender de implementações prontas da linguagem.
